This is a local, client-side only web "proxy" created by Maximus Weeseman (dinguschan.) Note that this is not a true proxy, because it is not routing your internet traffic through any external servers. Nonetheless, if you open any website through this file, it will not appear in your search history, not be affected by any blockers, trackers, or spyware (GoGuardian, Securly, etc.) The project works by using the fetch function in Javascript to fetch the HTML, CSS, and JS elements from a website, then displays it on this page. Keep in mind that this is a very basic example of this method, so following links in the website and non HTTPS websites are not supported. If you are interested, you can find the project on GitHub here. This will probably not be maintained, so I strongly urge you to use a better method to do your internet tomfoolery, and in the meantime, dinguschan out.
